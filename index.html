test test2z git init -> 깃 저장소 만들기 git remote add origin [ssh or https] -> 깃 저장소 연결 git add [file] -> 새로운 파일 깃 저장소에 추가 git status -> 현재 디렉토리 상황 보고 git commit -m "message" -> 커밋하고 기록 남기기 git push -> 깃 원격 저장소에 반영 git branch -> 깃 브렌치 상황 보고 git branch [new branch name] -> 브렌치 생성 git checkout [branch name] -> 브렌치 이동 git push origin [branch name] -> 깃 원격 브렌치에 반영 git pull -> 현재 브렌치의 코드 갖고 오기 git merge [branch name] -> 브렌치 병합 git branch -d [branch name] -> 브렌치 삭제(잘 사용하지 않음) git clone [http, ssh] -> 깃 원격 저장소의 레포지토리 복사
